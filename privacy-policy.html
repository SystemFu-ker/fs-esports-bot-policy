Privacy Policy for FS Esports Bot

Effective Date: February 9, 2026

FS Esports Bot respects your privacy. This Privacy Policy explains what information may be collected and how it is used while using the bot on Discord.

Information We May Collect

  • Discord User IDs
  • Server IDs
  • Channel IDs
  • Message content used for commands
  • Basic server activity required for bot functionality

How We Use Information

Collected information is only used to:

  • Provide bot features and commands
  • Manage moderation systems
  • Improve stability and performance
  • Prevent abuse or misuse

Data Sharing

We do not sell, trade, or share user data with third parties.

Data Storage

Most data is processed temporarily. Any stored data is kept only as long as necessary for bot functionality.

Security

Reasonable measures are taken to protect stored information from unauthorized access.

Third-Party Services

This bot operates through Discord and may rely on third-party services such as:

  • Discord API
  • YouTube API
  • Spotify API

Changes

This policy may be updated at any time without prior notice.

Contact

For support or questions, contact the FS Esports management team through Discord.
